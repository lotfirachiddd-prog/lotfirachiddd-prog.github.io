Guide d'utilisation

Comment lancer le logiciel

Suivre ces etapes dans l'ordre pour utiliser l'outil correctement.

Important : Le logiciel necessite les droits administrateur. Windows Defender peut afficher une alerte. C'est normal.

01

Telecharger le fichier

Telecharge le fichier depuis GoFile et sauvegarde-le dans un dossier dedie.

LagSwitchPro.exe (~74 Mo)
02

Ouvrir en administrateur

Clic droit sur le fichier et selectionne "Executer en tant qu'administrateur".

Clic droit > Executer en tant qu'admin
03

Configurer la touche

Clique sur "SET HOTKEY" et appuie sur ta touche de choix.

SET > Appuie sur F9 > DONE
04

Choisir le processus

Scan les processus actifs et selectionne ton jeu.

SCAN > Selectionne DayZ.exe
05

Regler la puissance

Commence entre 10% et 15% sur les jeux avec anti-cheat.

Slider : 10% - 15% recommande
06

Activer en jeu

Appuie sur ta touche configuree pour activer le lag. Relache pour stopper.

Appui = LAGGING | Relache = STABLE
